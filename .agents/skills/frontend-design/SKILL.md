# frontend-design

This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

The user provides frontend requirements: a component, page, application, or interface to build. They may include context about the purpose, audience, or technical constraints.

## Capabilities

- Modern UI/UX: Implement contemporary design patterns using CSS/Tailwind, Framer Motion, and React.
- Creative Direction: Choose typography, color palettes, and layouts that feel intentional and professional rather than default.
- Interaction Design: Add smooth transitions, micro-interactions, and animations that enhance the user experience without being distracting.
- Layout Mastery: Use CSS Grid and Flexbox to create robust, responsive, and interesting spatial arrangements.
- Data Visualization: Design beautiful charts, tables, and dashboards that communicate information clearly and elegantly.

## Guidelines

- Avoid standard "AI" styles (e.g., generic purple-to-blue gradients, rounded card layouts with excessive drop shadows).
- Use intentional white space, varied font weights, and refined color systems.
- Ensure all generated code is fully functional and uses the specified tech stack (React, Tailwind CSS, Framer Motion).
- Prioritize accessibility (aria-labels, semantic HTML) while maintaining high design quality.
